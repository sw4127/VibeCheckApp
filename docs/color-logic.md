# Colour Logic — principle of record (PM, 2026-07-17)

## The rule

Every colour choice on every surface must state, in writing:

1. **A psychological/physiological rationale** — what the wavelength does to the viewer and why
   that effect serves the screen's job. (PM's example: `#407AD8` is short-wave blue light, which
   suppresses excitement — suited to smoothing/calming contexts, e.g. a focused listening task.)
2. **A combination logic** — the harmony system that generated the supporting colours, not taste.
3. **One logic everywhere** — the same generative rules must govern all surfaces; a colour that
   can't cite the system doesn't ship.

This extends the design bar's "one accent in play" from restraint to *justification*. It applies
to product UI, share cards, OG images, and charts.

## The current system, argued under that rule (status: PENDING PM verdict)

**Gym gold `hsl(42 80% 62%)` on warm near-black `#0B0A08`** — the case for it:

- *Psychology:* the flagship instrument's subject IS prestige, and gold is the colour prestige
  wears — medals, laurels, gilt frames, "the gold standard." Long-wave warm light is arousing and
  appetitive; the palette is **deliberately seductive because seduction-by-label is what the test
  measures**. The mandatory debrief is the fool's-gold reveal — the aesthetic sets up the same
  trap the instrument does. A calming hue here would work *against* the measurement's drama.
- *Combination:* whole-field analogous harmony — every supporting tone lives in the warm band
  (hue 20°–52°: `FluidField` palette, dimmed link gold `hsl(42 45% 52%)`, muted text) over one
  near-black anchor; single accent in play per the design bar; luminance-floor transitions per
  the §27 system.
- *Consistency:* bias/gym surfaces (product, share card, OG image, write-up charts) all derive
  from this one band. Legacy spaces (football bright / music dark-drift) predate the pivot and are
  demoted doors.

**VERDICT (PM, 2026-07-17): option (b) — per-instrument hues under one system.**

- **Prestige Test keeps gold** *by its logic*: seduction-by-label is the subject; the palette
  sets the same trap the instrument does.
- **Delicacy battery gets a short-wave calm-focus blue** (`#407AD8` family) *by the same logic*:
  fine auditory discrimination wants suppressed excitement. This is the FIRST test of the rule —
  the Machine 02 build must open with its colour argument.
- **Every future instrument argues its hue the same way** before any UI ships. Shared
  saturation/luminance system + the near-black anchor keep it one product.

No new colours without an entry here.
